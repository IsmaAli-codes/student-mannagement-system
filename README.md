# Student Management System in C++

## Project Description
This is a simple Student Management System developed in C++ without using Object-Oriented Programming (OOP) concepts. The system uses arrays and functions to manage student records efficiently.

It allows users to add, display, search, and delete student information in a structured way using basic programming concepts.

---

## Features
- Add new student records
- Display all students

---

## Technologies Used
- C++
- Functions
- Loops
- Switch Case Statements

---

## How It Works
The program stores student data in parallel arrays:
- `ids[]` for student IDs  
- `names[]` for student names  
- `marks[]` for student marks  

A menu-driven system allows the user to perform different operations until they choose to exit.

---

## Project Purpose
This project was created for learning basic C++ programming concepts such as:
- Array handling
- Function usage
- Menu-driven logic
- Basic data management
